## 数据挖掘
1. 数据挖掘
2. 数据预处理：从数据中*检测*/*纠正*或删除损坏或不准确和不适用于模型的记录；让数据更适应模型，匹配模型需求
3. 特征工程：将原始数据转换为*更能代表预测模型的潜在问题的特征*的过程，通过提取相关的特征或者创造特征实现；可能面对的问题有：有*无关的特征*；特征之间有*相关性*；特征太多或太小；目的是：降低**计算成本**，提升**模型上限**

##  数据预处理
### 数据无量纲化
将不同规格的数据转换到同一规格，转换到某个特定的分布。可以提升模型的精度，防止防止取值范围大的特征对模型产生影响。但是决策树不需要**无量纲化**，可以处理任何的数据。

包括线性和非线性的处理方法
- 线性的无量纲化包括**中心化**和**缩放处理**


#### 数据归一化
将数据按照最小值中心化，再按极差缩放，Normalization（不是正则化）
$$
x^{*} = \frac{x-\min (x)}{\max (x) - \min (x)}
$$
归一化后的数据服从**正态分布**。

模块 `sklearn.preprocessing.MinMaxScaler`
实例化对象后可以通过 `fit` 以及 `transform` 将数据归一化；如果数据量比较大，则可以使用 `partial_fit` 归一化

```py
scaler = MinMaxScaler()
scaler.fit(X)

X_scaled = scaler.transform(X)

# 还愿
X_returned = scaler.inverse_transform(X_scaled)
X_returned
```

或使用 `numpy`
```py
X_array = np.array(X)
X_scaled_numpy = (X_array - X_array.min(axis=0)) / (X_array.max(axis=0) - X_array.min(axis=0))

X_scaled_numpy * (X_array.max(axis=0) - X_array.min(axis=0))  + X_array.min(axis=0)
```

#### 数据标准化
数据按均值中心化，再按标准差缩放，变成均值为 0，方差为 1 的正态分布。

```py
from sklearn.preprocessing import StandardScaler

scaler = StandardScaler()
scaler.fit(X)

scaler.mean_, scaler.var_

X_standard = scaler.transform(X)
X_standard
```

在标准化和归一化时，**空值**（NaN）在 `fit` 时会被直接忽略，而在 `transform` 后也会保持 NaN 的形式；只允许导入*二维的数组*。

一般来说选用的是标准化来进行特征缩放。`MinMaxScaler` 对异常值敏感，过大的异常值会使得其他相对较小的数据被压缩的很小，因此一般用 `StandardScaler`。而 `MinMaxScaler` 对于不涉及距离度量/梯度/协方差计算或者数据需要被压缩到特定区间时适用。

中心化只需要 pandas 广播减去某个数即可。其他 `sklearn` 中的缩放处理模块有：
- `MaxAbsScaler` 希望数据压缩，同时不损失稀疏性（不影响取值为 0 的个数）
- `RobustScaler` 在数据噪声和异常值比较大时可用

### 缺失值
使用 `sklearn.impute.SimpleImputer` 填充缺失值，包括 4 个重要的参数
- `missing_values` 缺失值时什么，默认是 `np.nan`
- `strategy` 填补缺失值的策略，有 `mean`，`median`，`most_frequent`，`constant`（需要配合 `fill_value`），*默认* `mean`
- `fill_value` 当参数 `strategy` 为 `constant` 时可用，表示填充的值，*默认 0*
- `copy` 默认为 True，表示创建一个副本
